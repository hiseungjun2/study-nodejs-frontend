1. 크롤링 / 크롤러
    Crawler (크롤러)
        조직적, 자동화된 방법으로 월드 와이드 웹을 탐색하는 컴퓨터 프로그램
        
    Seeds (씨드)
        정보를 탐색할 페이지를 의미 

    웹 크롤러, 웹 스크레퍼
        웹 스크레핑 : 웹 사이트를 원하는 정보를 자동으로 추출 & 수집하는 프로그램

    Robots.txt
        크롤러가 해당 사이트의 데이터를 수집하는 것을 허용할지 금지할지 알려주는 문서

2. 크롤링 모듈
    axios
        HTTP 요청과 HTML DOM Parser 사용하기
    Cheerio
        획득한 HTML 문서를 다루기 쉽게 파싱
    Puppeteer

        node.js 를 통해 크롬 브라우저를 실행

        마우스, 키보드, 터치스크린 등 코드를 통해 사람이 사용하는 것처럼 구현 가능
        타임라인 트레이싱, 스크린샷, PDF 다운로드, 확장 프로그램 테스트, 작업 자동화 가능
        SPA 크롤링과 Pre-rendered content 생성이 가능
